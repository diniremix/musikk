# Musikker
**Export your local playlists to Spotify, using the Web API and Python**

## Requeriments
- [python](http://wxpython.org/download.php)  2.7.x


## How to use
Clone this repo.

    git clone https://github.com/diniremix/musikker.git


```python
$ python musikker.py playlist.m3u
```
